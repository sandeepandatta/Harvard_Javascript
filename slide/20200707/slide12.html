CSS for Page Layout: Fixed Width

Define region widths based upon fixed sizes (pixels).

Use "div" elements to define areas and CSS to arrange them on the page.

<div id="wrapper">
  <div id="header"> ... </div>
  <div id="maincontent">
    ...
  </div>
  <div id="navigation">
    ...
  </div>
  <div id="footer"> ... </div>
</div>

    width: 750px;
